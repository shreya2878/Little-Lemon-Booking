# Little Lemon Booking

A responsive restaurant table booking website developed as part of the Meta Front-End Developer Professional Certificate.

## Features

- Responsive website design
- Table booking form
- User-friendly interface
- Clean and modern layout
- Mobile-friendly design

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

```
Little-Lemon-Booking/
│── index.html
│── css/
│── js/
│── images/
│── README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/shreya2878/Little-Lemon-Booking.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
